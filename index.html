Summary: Latitude vs. X

The purpose of the project was to analyze how weather changes as you get closer to teh equator. To accomplish the analysis we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset we used MathPlotLib to plot various aspects of teh weather vs. Latitude. Factors we looked at included temperature, cklouidiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis as well as explorations and descriptiios of any trends and correlations witnessed.
